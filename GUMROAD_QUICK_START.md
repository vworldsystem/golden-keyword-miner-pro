# Gumroad 빠른 시작 가이드

## 🎯 Gumroad는 어떻게 작동하나요?

### ✅ 자동으로 생성되는 것
- **제품 페이지 URL**: Gumroad가 자동으로 생성
- **결제 시스템**: 이미 구축되어 있음
- **구매자 이메일**: 자동으로 발송

### ⚠️ 직접 만들어야 하는 것
- **제품 정보**: 이름, 설명, 가격
- **제품 이미지/스크린샷**: 최소 1-2장
- **가격 설정**

## 📝 최소한으로 시작하는 방법

### 1단계: 기본 정보만 입력 (5분)

1. Gumroad 로그인 → **"Add a product"** 클릭

2. **필수 정보만 입력**:
   ```
   Name: Golden Keyword Miner Pro
   Price: $29 (또는 원하는 가격)
   Description: 
   AI 기반 키워드 분석 도구
   - 무제한 키워드 채굴
   - 롱테일 키워드 확장
   - SEO 최적화 분석
   ```

3. **"Save"** 클릭

4. **끝!** 제품 페이지가 자동 생성됨

### 2단계: 스크린샷 추가 (선택사항, 10분)

**더 잘 팔리게 하려면**:
- 앱 스크린샷 2-3장
- 간단한 설명 이미지

**없어도 판매는 가능합니다!**

## 🔗 제품 페이지는 어떻게 생기나요?

Gumroad가 자동으로 생성해주는 URL:
```
https://your-store.gumroad.com/l/golden-keyword-miner-pro
```

이 링크를:
- 소셜 미디어에 공유
- 블로그에 링크
- 이메일로 전송

## 💡 상세 페이지를 만들고 싶다면?

### 방법 1: Gumroad 내에서 작성
- 제품 편집 → "Description"에 마크다운/HTML 작성
- 이미지 추가
- 비디오 추가 (선택사항)

### 방법 2: 외부 랜딩 페이지
- 별도 웹사이트에 상세 페이지 제작
- Gumroad 결제 버튼만 연결

**하지만 처음에는 방법 1로 충분합니다!**

## 📸 스크린샷은 어떻게 찍나요?

### 필요한 스크린샷:
1. **메인 화면**: 키워드 입력 화면
2. **결과 화면**: 키워드 분석 결과
3. **Pro 혜택**: 업그레이드 모달

### 스크린샷 찍는 방법:
1. 앱 실행
2. `Windows + Shift + S` (스니핑 도구)
3. 원하는 영역 선택
4. 저장 후 Gumroad에 업로드
